## 개요
### 댕댕체크?
애견 미용사를 위한 애견 미용 예약 서비스입니다.  
대부분의 애견 미용실에서 예약 어플이 아닌 전화나 카카오톡을 통하여 예약을 받습니다.  
그래서 미용사들에게 예약을 받는데 불편함을 줄이고자 만들어졌습니다.
### 팀원 
- Beckend: 1명(최두영)

### 개발 기간
2025. 02 ~
### 기술 스택
- Java
- Spring Boot
- Spring Security
- Database: MySQL
- ORM: JPA, QueryDSL
- Authentification: JWT
- CI/CD: Github Actions
